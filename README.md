
# SS Studio | Premium Creative Web Agency

SS Studio is a high-performance digital agency specializing in E-commerce, Landing Pages, and custom UI/UX design. Built with a modern tech stack and integrated with Generative AI for project refinement.

## 🚀 Tech Stack

- **Framework**: [Next.js 15 (App Router)](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [Shadcn UI](https://ui.shadcn.com/)
- **Database & Auth**: [Firebase](https://firebase.google.com/)
- **AI Integration**: [Genkit](https://firebase.google.com/docs/genkit)
- **Icons**: [Lucide React](https://lucide.dev/)

## ✨ Key Features

- **AI Project Refiner**: A custom GenAI flow to help clients structure vague ideas into professional briefs.
- **Admin Dashboard**: Full CRUD management for Portfolio items and Blog posts.
- **Real-time Synchronization**: Powered by Firestore for instant updates.
- **Responsive Design**: Optimized for all devices with premium fluid animations.
- **Booking System**: Integrated appointment scheduling with WhatsApp fallbacks.

## 🛠️ Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/mdsajjad197/agency.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up environment variables**:
   Create a `.env` file in the root directory and add your Firebase and Genkit configuration.

4. **Run the development server**:
   ```bash
   npm run dev
   ```

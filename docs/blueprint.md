# **App Name**: StudioForge

## Core Features:

- Dynamic Portfolio Display with Filtering: Showcase SS Studio's diverse range of completed projects with intuitive categorization and filtering capabilities, allowing potential clients to easily explore relevant work.
- Interactive Service & Process Overview: Clearly articulate the services offered by SS Studio and outline the creative process from initial concept to final delivery, building transparency and trust.
- AI-Powered Project Concept Refiner: An AI tool to assist prospective clients in better defining their project ideas by interactively guiding them through key considerations and suggesting structured creative directions before submission.
- Streamlined Client Inquiry & Scheduling: Provide a comprehensive contact form for project inquiries, integrate WhatsApp for direct messaging, and allow seamless booking via Calendly for consultations.
- Secure Lead Storage & Management: Implement a backend system leveraging MongoDB to securely capture, store, and manage all incoming client leads and inquiries for efficient follow-up and relationship tracking.
- Fluid UI with Dark/Light Mode: Deliver a premium, minimalist user experience enhanced by smooth scroll animations, elegant transitions, and a user-toggleable dark/light theme for aesthetic personalization.
- Mobile-First Responsive & Performant Design: Ensure the website provides an optimal and high-performance viewing experience across all devices, prioritizing responsiveness and fast loading times for mobile users.

## Style Guidelines:

- Primary Color: A sophisticated deep blue (#174CB3), conveying trust and professionalism for interactive elements and key accents within a light scheme.
- Background Color: A very light, almost neutral bluish-grey (#F2F5F9), providing a clean and minimal canvas that gently harmonizes with the primary color.
- Accent Color: A bright, vibrant cyan (#2EDAFF), used sparingly for impactful call-to-actions, highlights, and subtle creative flair, ensuring high contrast and modern appeal.
- Headline Font: 'Space Grotesk' (sans-serif) for its modern, slightly technical, and professional feel, lending impact to titles.
- Body Font: 'Inter' (sans-serif) for exceptional readability, clean lines, and neutral aesthetic, suitable for extensive content and descriptions.
- Utilize minimalist, crisp line-art icons that maintain visual clarity and contribute to the clean, modern aesthetic.
- Employ a spacious, asymmetric grid layout with ample white space to emphasize content and foster a premium, organized feel. Adhere strictly to mobile-first principles.
- Integrate subtle, smooth transitions and micro-interactions (e.g., hover effects, content reveals) to enhance the user experience without distracting, aligning with the requested 'smooth animated user interfaces'.
